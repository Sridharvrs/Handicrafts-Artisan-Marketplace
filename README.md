# Karigar — Handicrafts & Artisan Marketplace

A 5-page handmade static website. Pure HTML, CSS, and JavaScript. No build step.

## Pages
- `index.html` — Home (editorial hero + bento categories + marquee)
- `shop.html` — Shop (sticky filters + product grid)
- `artisans.html` — Artisans (zigzag maker profiles)
- `story.html` — Our Story (timeline + values)
- `contact.html` — Contact (split-screen form)

## Run
Just open `index.html` in a browser, or serve the folder:
```
python3 -m http.server 8000
```

## Design system
All tokens (colors, fonts, spacing) live in `css/style.css` under `:root`.
Palette: cream / terracotta / forest / ink. Typography: Fraunces + Inter + JetBrains Mono.
